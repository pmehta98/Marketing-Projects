## 📁 Projects

### 🛒 Basket Analysis

- **Folder:** [`Basket Analysis`](./Basket%20Analysis)
- **Objective:** Identify top product pairs for cross-promotion in a grocery store to maximize cross-sell opportunities
- **Methodology:**
  - Market Basket Analysis
  - Association Rules Mining
  - Support, Lift, and Confidence metrics
  - TF-IDF analysis for product segmentation
- **Key Insights:**
  - Highlighted five high-performing product pairs based on Support, Lift, and Confidence
  - Recommended strategic bin placements and promotional campaigns to enhance shopping experience
  - Excluded irrelevant or impractical pairs based on real-world limitations (e.g., legal restrictions)
- **Recommendations:**
  - Cross-promotional displays for complementary products
  - Targeted discount bundles
  - Strategic shelf placement optimization

---

### 💻 Online Advertising Optimization

- **Folder:** [`Online Advertising`](./Online%20Advertising)
- **Objective:** Reallocate advertising budgets across multiple platforms (Reddit, LinkedIn, Google, Facebook) to maximize ROI
- **Platforms Analyzed:**
  - Reddit
  - LinkedIn
  - Google Ads
  - Facebook
- **Key Metrics:**
  - Return on Investment (ROI)
  - Click-Through Rate (CTR)
  - Cost Per Click (CPC)
  - Conversion Rate
- **Key Insights:**
  - **Reddit** outperformed all platforms with **179% ROI** and **2.07% CTR**
  - LinkedIn and Facebook showed lower returns, requiring improved ad targeting
  - Provided data-driven budget reallocation strategy
- **Recommendations:**
  - Increase Reddit advertising budget by 40%
  - Reduce underperforming platform spend
  - Implement A/B testing for ad creative optimization

### 💊 Pharmaceutical Marketing Assessment

- **Folder:** [`Pharmaceutical Marketing`](./Pharmaceutical_Marketing_Instructions)
- **Objective:** Evaluate marketing programs to optimize prescription lift across physician segments
- **Programs Evaluated:**
  - Loyalty Cards
  - Speaker Programs
  - Medical Conferences
  - Direct Mail Campaigns
  - Digital Engagement
- **Segmentation:**
  - High-Prescribing Physicians (HRPs)
  - Medium-Prescribing Physicians (MRPs)
  - Low-Prescribing Physicians (LRPs)
- **Key Insights:**
  - **Loyalty Cards** and **Speaker Programs** provided highest prescription lift by Week 4
  - Regional analysis revealed geographic performance variations
  - HRPs showed highest response to targeted programs
- **Recommendations:**
  - Reallocate resources to high-performing marketing programs
  - Develop metrics for long-term conference engagement tracking
  - Implement personalized physician outreach strategies

## 🛠️ Setup

### Prerequisites

- Python 3.8 or higher
- R (for statistical analysis)
- Excel or Google Sheets
- Jupyter Notebook

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pmehta98/Marketing-Projects.git
   cd Marketing-Projects
   ```

2. **Navigate to a specific project:**
   ```bash
   cd "Basket Analysis"
   # or
   cd "Online Advertising"
   # or  
   cd "Pharmaceutical_Marketing_Instructions"
   ```

3. **Install required packages:**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
